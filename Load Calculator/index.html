SINGLE-POLE: A single pole breaker is one that takes up 1 slot. See pictures above for example.

DOUBLE-POLE: A double pole breaker is one that takes up 2 slots. See pictures above for example.

QUAD: A quad breaker is when 2 Double-Pole breakers are combined into 1. In the example above it shows a 20A/50A Quad Breaker. This means we have (1) 20A Double-Pole & (1) 50A Double-Pole Breakers.

TANDEM/TWIN: A Twin/Tandem breaker is when 2 Single-Pole breakers are combined into 1. In the example above it shows a 20A/20A Twin Breaker. This means we have (2) 20A Single-Pole

SMALL LOADS: A small load is any breaker 25A or less.

SPECIAL LOADS: Any breaker greater than 25A. You will need to fill out a line for each breaker greater than 25A and select whether it is Single or Double-Pole

HEAT/AC UNITS

HEATER: These are labeled on the MSP Panel. If you have multiple, then you should ONLY use the biggest breaker. If there are no breakers labeled heater or no breakers labeled at all, leave this section blank.

AC UNITS: These are also labeled on the MSP Panel. If the breakers are not labeled then use the biggest breaker you can find. If you have multiple AC Units, use the biggest one for the AC Units section, and use the other(s) in the special loads section. If all loads are small loads then leave the AC Units section blank. Make sure you do not include this breaker again in the Specials Loads Section!

BREAKER LABELS: Breakers Labels will be located right next to the breakers or on a labels sheet on the Main Service Panel Door.

PV SYSTEM INTERCONNECTION CALCULATION - "120% RULE" NEC 705.12(D)(2)

Formula: Main Panel Busbar Rating x 120% - Main Breaker Rating

Current Panel Rating: A/A      Proposed PV Breaker: A

A (busbar rating) x 1.2 = A - A (main breaker) = A (solar allowance)

A (solar allowance) > A (proposed PV breaker)

120% Rule: OKAY
